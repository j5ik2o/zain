# Suggested Commands
- `ls -la` : list files
- `rg --files` : fast file listing
- `rg "pattern" path` : fast text search
- `sed -n '1,200p' <file>` : inspect file content
- `sbt compile` : compile all aggregated modules
- `sbt test` : run tests for aggregated modules in root project
- `sbt "project core" test` : run tests for specific module
- `git status --short` : check worktree changes
