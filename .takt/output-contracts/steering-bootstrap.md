# Steeringブートストラップ出力契約

## 必須セクション

```markdown
## モード

Bootstrap

## 生成ファイル

- product.md: [プロダクトの概要を1行で]
- tech.md: [主要技術スタックを1行で]
- structure.md: [コード構成パターンを1行で]

## 抽出パターン / 初期方針

[コードベースから検出した主要パターンの要約、またはグリーンフィールドの場合はタスク入力・テンプレート由来の初期方針]

## 次のステップ

[Source of Truthとしてレビュー・承認を促す]
```

## 品質基準

| 基準 | 判定 |
|------|------|
| 3つのコアファイルが生成されている | 必須 |
| パターンまたは初期方針が記録されている（リストではない） | 必須 |
| セキュリティ情報が含まれていない | 必須 |
| 各ファイルが100-200行以内 | 推奨 |
| テンプレートの構造に従っている | 必須 |
