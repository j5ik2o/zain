# Steeringシンク出力契約

## 必須セクション

```markdown
## モード

Sync

## 変更内容

- [ファイル名]: [変更の概要]
- ...

## コード乖離

- [乖離の説明]
- ...

## 推奨事項

- [追加すべきカスタムsteeringファイルの提案等]
- ...
```

## 品質基準

| 基準 | 判定 |
|------|------|
| 検出された乖離が具体的に記述されている | 必須 |
| ユーザーセクションが保持されている | 必須 |
| 更新が追加的である | 必須 |
| セキュリティ情報が含まれていない | 必須 |
| updated_atタイムスタンプが付与されている | 推奨 |
