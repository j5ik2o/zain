# カスタムSteering出力契約

## 必須セクション

```markdown
## 作成ファイル

- .kiro/steering/{name}.md

## ベース

- テンプレート: [テンプレート名 または「スクラッチ生成」]
- 分析対象: [分析したディレクトリやファイルパターン]
- 抽出パターン: [検出した主要パターン]

## コンテンツ概要

- [セクション1の概要]
- [セクション2の概要]
- ...

## 次のステップ

[レビューとカスタマイズの促し]
```

## 品質基準

| 基準 | 判定 |
|------|------|
| 1ファイル1トピックである | 必須 |
| パターンが記録されている（リストではない） | 必須 |
| セキュリティ情報が含まれていない | 必須 |
| 100-200行以内に収まっている | 推奨 |
| コアsteering（product/tech/structure）と重複していない | 必須 |
| 具体的なコード例が含まれている | 推奨 |
| テンプレート構造に従っている（テンプレートがある場合） | 必須 |
